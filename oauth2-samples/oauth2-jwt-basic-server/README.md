# Spring Cloud Security: OAuth2 JWT Server

## Checkout
```bash
] git clone https://github.com/lllabs/l2labs-spring-cloud-samples.git
```

## Run
```bash
] cd PROJECT_ROOT
] ./gradlew :oauth2-samples:oauth2-jwt-basic-server:bootRun
```
